## IT Asset Management System



**Live Demo:** https://it-asset-management-frontend.onrender.com  

**Login:** admin@company.com
**Password:** admin123



**Features**

---

* Login \& secure authentication
* Dashboard with live asset stats
* Add/View/Delete assets (MacBook Pro, Dell Monitor in DB)
* Assign assets to employees  
* Professional Cyber Operations dark theme


  

**Tech Stack** 

---

* Frontend : React + Bootstap 5
* Backend: Node.js + Express + MongoDB Atlas
* Auth: JWT + bcrypt
* Deployment: Render.com (free)
  



###### **Quick Start (Local)**



* **Backend:**


cd backend

npm install

npm run dev


* **Frontend:**



npm install

npm start
