## Features



Login + Dashboard

Add/View/Delete assets (MacBook + Monitor in DB)  

Assign assets to employees

Cyber Operations dark theme







## Tech Stack



Frontend: React.js + Bootstrap 5

Backend: Node.js + Express + MongoDB Atlas

Deployment: Render.com (free)





## Quick Start



### Backend



cd backend

npm install

npm run dev



### Frontend



npm install

npm start
